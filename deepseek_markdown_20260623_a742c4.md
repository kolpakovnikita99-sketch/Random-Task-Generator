# Random Task Generator

**Автор:** Иванов Иван Иванович  
**Версия:** 1.0

## Описание
Приложение с графическим интерфейсом (Tkinter) для генерации случайных задач из списка. Поддерживает фильтрацию по категориям, сохранение истории в JSON и добавление новых задач.

## Функциональность
- Генерация случайной задачи из предопределённого или пользовательского списка.
- Фильтрация по категориям: учёба, спорт, работа.
- Отображение текущей задачи и истории всех сгенерированных задач.
- Добавление новых задач с указанием категории.
- Сохранение всех данных в файл `tasks_history.json` для сохранения между сеансами.

## Установка и запуск
1. Убедитесь, что установлен Python 3.6+.
2. Скачайте файл `task_generator.py`.
3. Запустите: `python task_generator.py`.

## Пример использования
1. Запустите программу.
2. Нажмите «Сгенерировать задачу» — появится случайная задача.
3. Используйте фильтр для отображения только задач определённой категории.
4. Введите новую задачу и выберите категорию, затем нажмите «Добавить».
5. Все сгенерированные задачи сохраняются в истории и в JSON-файле.

## Тесты (пример)
- При пустом списке задач кнопка генерации показывает предупреждение.
- Добавление пустой задачи не допускается.
- Фильтр корректно отображает только задачи выбранной категории.
